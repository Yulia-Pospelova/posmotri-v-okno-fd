# Video Search Interface

Проект представляет собой интерфейс поиска видео, реализованный на HTML и CSS.  
Выполнен в рамках учебной практики по верстке интерфейсов.

## Описание

Страница включает:

- Видео-превью
- Форму поиска с кастомными чекбоксами
- Список карточек видео
- Кнопку загрузки дополнительных элементов

Реализованы интерактивные состояния элементов интерфейса.

## Технологии

- HTML5
- CSS3
- Flexbox
- Методология БЭМ
- Accessibility-паттерн `.visually-hidden`
- Ограничение текста через `-webkit-line-clamp`
- Prettier (автоформатирование)
